# GitHub Action Python Toolkit

Python module for writing GitHub Actions.


## Usage

### Build

```bash
# Get the repo
$ git clone https://github.com/boschglobal/action.toolkit.git
$ cd action.toolkit

# Build
$ make

# Test
$ pip install -r requirements.txt
$ make install
$ make test
```
